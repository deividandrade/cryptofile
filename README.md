# 🔐 CryptoFile

CryptoFile é uma ferramenta simples desenvolvida em **Java** para **criptografar** e **descriptografar** arquivos.  
O objetivo do projeto é permitir proteger arquivos sensíveis como documentos, imagens e outros tipos de dados através de criptografia.

## ✨ Funcionalidades

- 🔒 Criptografia de arquivos
- 🔓 Descriptografia de arquivos criptografados
- 📁 Suporte para qualquer tipo de arquivo
- ⚡ Implementação simples utilizando as bibliotecas padrão do Java

---

## 📦 Requisitos

- Java 8 ou superior

Verifique sua versão:

```bash
java -version
```

# 🚀 Como executar
1. Clonar o repositório
```
git clone https://github.com/deividandrade/cryptofile.git
cd cryptofile
```
2. Compilar o projeto
```
javac CryptoFile.java
```
4. Executar o programa
```
java CryptoFile
```
# 🔐 Como funciona

O CryptoFile utiliza as bibliotecas de criptografia padrão do Java para:

Ler o arquivo de entrada

Gerar uma chave criptográfica baseada na senha

Aplicar o algoritmo de criptografia

Salvar o arquivo criptografado

O processo de descriptografia utiliza a mesma senha para restaurar o conteúdo original.
